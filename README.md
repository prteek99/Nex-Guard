# 🛡️ NexGuard: Advanced Phishing Defense & Education

NexGuard is a premium, AI-powered educational platform designed to empower users against digital threats. It combines interactive learning, real-world simulations, and a sleek, modern UI to make cybersecurity accessible to everyone.

**🌐 Live Demo:** [nex-guard-chi.vercel.app](https://nex-guard-chi.vercel.app)

---

## 🚀 Key Features

### 🎓 Safety Academy
Deep-dive learning modules focused on email phishing, dangerous links, and red flags. Features rich visual content and interactive elements.

### 🕹️ Spot the Phish (Simulator)
An interactive gallery of real-world scenarios. Users are challenged to identify legitimate vs. fraudulent communications across Email and SMS channels with detailed explanations for every result.

### 🧠 Interactive Quiz Center
A comprehensive 40-question quiz engine that tests knowledge on various cybersecurity topics, providing immediate feedback and score tracking.

### 🔍 Real-time Analysis
Sleek, premium UI for analyzing threats with micro-animations and clear risk assessment data.
